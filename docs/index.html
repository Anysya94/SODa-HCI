Abstract

The purpose of this project is to offer a proper data visualization solution for 2 different datasets. These datasets are public and available as follows :
  • U.S. Department of Agriculture's Plants Database - A public dataset about plants
  • Good Books Dataset - A public dataset containing information about goodreads books

U.S. Department of Agriculture's Plants Database

The Complete PLANTS Checklist is nearly 7 MB and includes Symbol, Synonym Symbol, Scientific Name with Authors, National Common Name, and Family. Fields in this text file are delimited by commas and enclosed in double quotes.

Good Books Dataset

This dataset contains ratings for ten thousand popular books. Generally, there are 100 reviews for each book, although some have less - fewer - ratings. Ratings go from one to five.

I process and analyse data from books.csv which contains metadata for each book (goodreads IDs, authors, title, average rating, etc.). The metadata have been extracted from goodreads XML files, available on https://www.kaggle.com/zygmunt/goodbooks-10k/data.

Then I made several statistics on the number of books published by year and the number of books written by each author over time and represent each data set using Chart.js library with different kind of charts. (DONUT, PIE AND BAR)
